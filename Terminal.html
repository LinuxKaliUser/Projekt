Ich verwende das Terminal immer, wenn ich versuche etwas zu Hacken oder wenn ich Linux benutze. Manche Applikationen in Kali kann man nur mit dem Terminal benutzen und es ist praktischer das Terminal zu verwenden, als mühsam die Applikation selber auf dem Desktop zu öffnen.

Man kann auch im Terminal Scripts in Bash schreiben und verwenden, was ich selten mache, aber mir sicher vornehmen will. Es gibt verschiedene Pfade, die man mit dem Terminal wechsel oder verändern kann, aber der Default ist immer der “Kali” Ordner Pfad, wenn man das Terminal öffnet.

Wenn man Applikationen oder Modul im Kali verwenden will, muss man es extern im Internet holen, wenn es nicht mit dem Betriebssystem darauf ist. Es ist extrem schwer, es einfach mit dem Interface/Oberfläche eine Applikation herunterzuladen. Zum Glück steht es meistens auf der Webseite von der Applikation, die man herunterladen will immer den Terminal Code schon. Dann muss man es mit Sudo (Super User do) ausführen lassen und nachdem hat man die externe Applikation schon installiert.
